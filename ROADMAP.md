ROADMAP.md — Climate & Phenology Journal
Overview
This roadmap outlines the development path for the Climate & Phenology Journal — a long‑term, community‑driven tool for recording weather, phenology, and seasonal change in any region.
The goal is to build a simple, durable, offline‑capable system that fills the gaps between satellite data, sensor networks, and lived experience.

This roadmap is intentionally flexible. It will evolve as contributors join, regions participate, and new needs emerge.

Establish the core structure of the project.

1. Repository Setup
Add LICENSE (Apache 2.0)

Add CODE_OF_CONDUCT.md

3. Design & UX
