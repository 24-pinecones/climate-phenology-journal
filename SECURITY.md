Security Policy
  Thank you for helping keep this project safe for everyone.

Reporting a Vulnerability
  If you discover a security issue, please report it responsibly.

To report a vulnerability:
  Open a private security advisory using GitHub’s “Report a vulnerability” feature
  (found under the “Security” tab of this repository)

  Or email the project maintainer directly if you prefer private communication

Please include:
  A clear description of the issue
  Steps to reproduce (if applicable)
  Any suggestions for remediation

We will review the report promptly and work with you to address the issue.

Supported Versions
  This project is in early development.
  All versions are currently supported for security updates.

Responsible Disclosure
  Please do not publicly disclose vulnerabilities until they have been reviewed and addressed.
  We appreciate your help in keeping the project safe for all contributors and users.
