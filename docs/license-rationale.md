# Why This Project Uses the Apache 2.0 License

The Climate & Phenology Journal is intended as a long‑term public‑good tool that supports emergency services, researchers, land managers, and communities. To ensure the project is safe, stable, and collaboration‑friendly, we selected the Apache License 2.0.

## Key Reasons

### 1. Clear Legal Permissions
Agencies and organizations can:
- use the software  
- modify it  
- integrate it into internal systems  
- build tools on top of it  

without needing special agreements.

### 2. Patent Protection
Apache 2.0 includes explicit patent grants, reducing legal uncertainty for:
- wildfire agencies  
- flood and emergency services  
- universities  
- corporations  

### 3. Encourages Broad Collaboration
The license is widely used in:
- government open‑source projects  
- scientific and research software  
- corporate open‑source contributions  

### 4. Long‑Term Stability
Apache 2.0 is designed for projects that:
- evolve over decades  
- involve many contributors  
- support public safety and environmental monitoring  

This aligns with the long‑term vision of building a shared climate and phenology record for any region.
